电路维修

达达是来自异世界的魔女，她在漫无目的地四处漂流的时候，遇到了善良的少女翰翰，从而被收留在地球上。

翰翰的家里有一辆飞行车。

有一天飞行车的电路板突然出现了故障，导致无法启动。

电路板的整体结构是一个 $R$ 行 $C$ 列的网格 $（R,C≤500）$，如下图所示。

每个格点都是电线的接点，每个格子都包含一个电子元件。

电子元件的主要部分是一个可旋转的、连接一条对角线上的两个接点的短电缆。

在旋转之后，它就可以连接另一条对角线的两个接点。

电路板左上角的接点接入直流电源，右下角的接点接入飞行车的发动装置。

达达发现因为某些元件的方向不小心发生了改变，电路板可能处于断路的状态。

她准备通过计算，旋转最少数量的元件，使电源与发动装置通过若干条短缆相连。

不过，电路的规模实在是太大了，达达并不擅长编程，希望你能够帮她解决这个问题。

注意：只能走斜向的线段，水平和竖直线段不能走。

对于每组测试数据，在单独的一行输出一个正整数，表示所需的缩小旋转次数。

题解

这个题还是很有意思的。核心思路是，把需要旋转的边看作边权为 $1$，不需旋转的边看作边权为 $0$，求一下最短路。

需要考虑的是，有没有可能旋转一个边两次呢？这是一个连通的电路，旋转两次的话电路是通不了电的。可以保证最多只旋转一次，这是因为我们只能达到横纵坐标和为偶数的点（想想为什么）。

然后，由于 $500 \times 500 = 250000$，朴素迪杰斯特拉是一定不行的，（堆优化也许可以hhhh）。 对于 $01$ 边权的最短路，有双端队列bfs算法，若边权为 $0$， 插到队列前，若边权为 $1$ ，插到队列后。可以证明所求满足最短路，这里不赘述证明方法。

由于本题既需要考虑点，又需要考虑方格中的边，所以代码细节上需要注意。

$O(n)$复杂度，可以AC。

数据范围

$1≤N,M≤11$

输入样例：

输出样例：

题解

我们可以使用状态压缩dp考虑这道题目。

首先，我们并不需要知道所有长方形是怎么排列的。要统计所有数的排列数，我们只需要知道横向或竖向的长方形的排列数即可。这里，我们统计横向长方形的排列。

我们定义状态 $f[i,j]$ 为第 $i$ 列横方块的排列方式。所谓一个横方块在 $i$ 列的意思是，其第一个格子在 $i$ 列。至于 $j$ ，我们把 $j$ 看作一种描述排列的方式。把 $j$ 转换成二进制数，这个数第 $k$ 位上为 $1$ 表示第 $k$ 行放了一个方块。

举个例子，如果一共 $5$ 行，则数字 $5 = (00110)_2$ ，表示第二列和第三列有方块。

我们考虑如何从其他状态转移得到 $f[i,j]$ 。不难想到，要安排第 $i$ 列，我们只需要考虑第 $i - 1$ 列就行了。比如说，如果一共五行，第 $i-1$ 列的第二、三行有方块，那么我们便可以在第 $i$ 行的一、四、五行随意安排方块。

当然，除此之外，要想方块排布合法，我们必须保证能放竖着的方块。这必须保证，每一列上横方块的排布中，连续空位必须为偶数个。继续说上面的例子，如果我们把横方块放在一、四，那么第五块便空了出来，但这块又放不了竖方块，所以这种方法是不合法的。实际上，在这个例子中，只有在五上放方块这一个情况是合法的。

对于某一种排列是否合法，我们可以预处理出所有情况。

对于如何找到合法的排列，我们可以对相邻两列暴力枚举，即写两个循环。由于题目中数据量为 $11$ ，故可以保证不TLE。

至于我们最后需要输出什么，很显然，我们需要的是，所有最后一列的横方块，都是第二个格子在最后一列，否则就凸出来了。这种情况对应的状态数量是 $f[m,0]$ 。

最终耗时为 $O(m4^n)$

最短编辑距离

给定两个字符串 $A$ 和 $B$，现在要将 $A$ 经过若干操作变为 $B$，可进行的操作有：

  1. 删除–将字符串 $A$ 中的某个字符删除。
  2. 插入–在字符串 $A$ 的某个位置插入某个字符。
  3. 替换–将字符串 $A$ 中的某个字符替换为另一个字符。

现在请你求出，将 $A$ 变为 $B$ 至少需要进行多少次操作。

输入格式

第一行包含整数 $n$，表示字符串 $A$ 的长度。

第二行包含一个长度为 $n$ 的字符串 $A$。

第三行包含整数 $m$，表示字符串 $B$ 的长度。

第四行包含一个长度为 $m$ 的字符串 $B$。

字符串中均只包含大写字母。

输出格式

输出一个整数，表示最少操作次数。

数据范围
$1≤n,m≤1000$
输入样例：

输出样例：

题解

状态定义：$f[i, j]$ 为将 $A$ 的前 $i$ 个字符串转化成 $B$ 的前 $j$ 个字符串所需的最少操作数。

我们考虑对 $f[i]$ 进行的所有可能的操作。

  1. 删除操作。我们可以先做 $f[i - 1, j]$ 对应的操作， 最后把多余的 $a[i]$ 删掉即可。此时 $f[i, j] = f[i - 1, j] + 1$ 。
  2. 增加操作。我们可以先做 $f[i, j - 1]$ 对应的操作，最后再在串后面加上 $b[j]$ 即可。此时 $f[i, j] = f[i, j - 1] + 1$ 。
  3. 修改操作。我们可以先做 $f[i - 1, j - 1]$ 对应的操作。如果 $a[i] == b[j]$， 则直接结束。否则，将 $a[i]$ 改成 $b[j]$ 。此时 $f[i, j] = f[i - 1,, j - 1] + 1 \ or \ 0$。

由于是求最小值，故上述三种方案中求最小值。

综上，得状态转移方程：

$$f[i, j] = min(f[i - 1, j] + 1, f[i, j - 1] + 1, f[i - 1, j - 1] + 1 \ or \ 0)$$
