常用Go语言标准库

发表于 2022-07-05

记录了一些常用Go语言标准库的用法。

flag

flag包实现了解析命令行标识的功能。

import "flag"

使用flag.String() flag.Bool() flag.Int() 声明三种类型的命令行标识。

以flag.String()为例

共有三个参数，第一个参数name表示了flag名，即运行程序时时指明的-name；第二个参数value表示flag默认值，第三个参数usage则写明了标识的作用。其在使用-h时展现，如：

上面三个函数返回对应数据类型的指针。在主程序中，调用flag.Parse()进行参数绑定，使指针指向的数据获得标识指示的值。也可以直接使用Var函数绑定变量，其直接赋值一个变量而非指针。

text/template

text/template包实现了基于数据+模板批量生成文本输出的功能。

模板文本使用UTF-8的字符串。而模板数据或控制符使用{{` `}}包围，称作“Actions”。传入的数据作为数据结构使用，.号标识了当前数据。

pipeline

pipeline是产生数据的操作，比如 {{.}}，{{.Name}}，{{funcname args}}，

使用|可以在pipeline中传输数据，如{{len(.) | println}}向println函数传入参数len(.)。
